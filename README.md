## X.COM (twitter) 클론 코딩

현 X.COM 구 Twitter 를론 코딩함
Nextjs App router 공부를 위해 기본적인 UI 및 기능 클론 코딩
msw(mockServiceWorker)를 사용하여 가짜 api 구성하여 통신
임의의 데이터는 faker-js를 사용

## 사용 스택

React 18
Next14 App router
css.module
next-auth
react-query
faker-js
msw
express

## 실행 방법

```bash
# next front server run
npm run next
# msw run
npm run mock
```
